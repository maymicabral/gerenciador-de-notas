# Gerenciador de Notas 📝

Este é um mini projeto em Node.js para gerenciar notas via terminal.

## 🚀 Como usar?

### Adicionar uma nota:
```sh
node app.js add "Título da Nota" "Conteúdo da Nota"

### Listar uma nota:
```sh
node app.js list

### Deletar uma nota:
```sh
node app.js delete "Título da Nota"

## 📌 Tecnologias usadas:
Node.js
Módulo fs (File System)
